
## Executando

Iniciar a aplicação e criar os contêineres

```bash
$ git clone https://github.com/rtmedical/rh-react.git
$ cd rh-react
$ docker-compose up
```

Interromper e remover os contêineres, redes, volumes

```bash
$ docker-compose down
```
